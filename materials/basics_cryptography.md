# Cryptography Decryption Sandbox Lab

## Overview
In this lab, you will learn how to analyze basic encrypted ciphertexts and retrieve hidden flags using decryption parameters.

## Core Concepts
- **Symmetric Encryption**: Uses the same key for both encryption and decryption (e.g., AES-256).
- **Flag Format**: NCAS flags follow the structure: `NCAS{flag_details}`.

## Lab Instructions
1. Run `ls` to list the files in the directory.
2. View the contents of `secret.txt` using the command: `cat secret.txt`.
3. Extract the cipher code and input it into the flag validator to earn +200 XP.
