Web Service Scan & Diagnostics Policy

Security Diagnostics Compliance Policy

1. Authorized Diagnostic Scope

This CyberShield Web Service is an educational, diagnostic resource designed to support network status lookups and threat analysis. Active vulnerability discovery, unauthorized brute forcing, or port scanning of assets without specific authorization is strictly prohibited. All domain lookups, WHOIS requests, and DNS resolutions utilize passive, public APIs or standard socket lookups.

2. Privacy & User Data Protection

- File Hashing utility generates cryptographic signatures locally in the user's browser via the HTML5 Web Crypto API. File contents are never transmitted to the host server. - Secret key phrases used in the AES Cryptography Tool are processed locally and are not cached or stored on the backend.

3. Security Incident Reporting

Any user who detects active phishing campaigns, spoofing attempts, or security policy violations targeting our institutional networks must report the event immediately using the Report Cyber Threat portal. The Admin Console tracks all submissions to coordinate mitigation steps.
